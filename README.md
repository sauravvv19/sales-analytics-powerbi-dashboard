# powerbi-sales-dashboard

📌 Project Overview

This project presents an interactive Sales Dashboard built using Power BI to analyze business performance across revenue, profit, customers, and products.
The dashboard provides a comprehensive view of key metrics and enables users to explore trends, patterns, and insights through dynamic visualizations.

📌 Problem Statement

Businesses often struggle to track sales performance and identify key growth areas due to scattered and unstructured data.
The objective of this project is to:

Monitor overall business performance
Identify top-performing products and customers
Analyze sales trends over time
Enable data-driven decision-making


📈 Key Insights
Generated ₹24.9M total revenue with ₹10.5M profit
Observed a consistent upward trend in weekly revenue
Accessories category contributed the highest number of orders
Identified top customers contributing significantly to revenue
Detected products with high return rates, indicating improvement areas



🛠 Tools Used
Power BI – Dashboard development & data visualization
DAX (Data Analysis Expressions) – Measures and calculations
